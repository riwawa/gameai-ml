# Game AI Project 

AI-powered game system that simulates player behavior using Java and applies Machine Learning (Python) with K-Means clustering to dynamically adapt game difficulty in real time.

## 🧠 Tech Stack
- Java (Game simulation)
- Python (Machine Learning)
- Scikit-learn (K-Means)
- FastAPI (API layer)

## ⚙️ Features
- Player behavior simulation
- Automatic dataset generation
- Unsupervised learning (K-Means clustering)
- Dynamic difficulty adjustment

## 🚀 Goal
To simulate adaptive game intelligence based on player behavior.
